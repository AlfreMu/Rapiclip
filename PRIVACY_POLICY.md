# Política de Privacidad / Privacy Policy

## 🇪🇸 Español

Rapiclip no recopila, almacena, vende ni transfiere datos personales de los usuarios.

La extensión únicamente interactúa con la página activa de videos de YouTube para:

- leer el tiempo actual de reproducción seleccionado por el usuario
- detectar clics del usuario en los botones de la extensión
- generar un enlace compartible con los tiempos de inicio y fin del clip

No se recopila información personal, historial de navegación, contraseñas, datos financieros, ubicación ni comunicaciones privadas.

Todo el procesamiento se realiza localmente en el navegador del usuario.

El enlace generado solo abre la página pública de Rapiclip con el ID del video y los timestamps seleccionados.

## 🇺🇸 English

Rapiclip does not collect, store, sell, or transfer personal user data.

The extension only interacts with the active YouTube video page in order to:

- read the current playback time selected by the user
- detect user clicks on the extension buttons
- generate a shareable clip link with selected start and end timestamps

No personal information, browsing history, passwords, financial data, location data, or private communications are collected.

All processing happens locally in the user's browser.

The generated link only opens the public Rapiclip web page with the selected video ID and timestamps.
